1. Definition

Hebun is the ontological foundation of the Zanistarast system.

It defines existence not as isolated entities,
but as a layered, interconnected structure.

---

2. Core Principle

Existence is structured in layers.

Each layer:
- includes the lower layer
- does not violate lower-layer laws
- expands system capacity

---

3. Structural Rule

Upper Layer ⊇ Lower Layer

If a higher layer contradicts physical laws:
→ it is invalid

---

4. Functional Meaning

Hebun transforms existence from:
random → structured  
isolated → interconnected  
static → dynamic  

---

5. Layer Model

D1 → Physical  
D2 → Biological  
D3 → Mind  
D4 → Law  
D5 → Ethics  

Each layer:
- emerges from pressure of complexity
- increases system capacity

---

6. Transition Logic

New layers appear when:

Complexity (K) exceeds capacity (T)

K / T > 1 → new layer required

---

7. Failure Condition

If layering breaks:

- system becomes unstable
- contradictions increase
- collapse becomes inevitable

Hebun is not a concept.

All systems must align with it
