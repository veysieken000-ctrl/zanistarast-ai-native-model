Version: 1.0

---

## 1. PURPOSE

This document defines the canonical repository architecture for the AI-native scientific representation of the Zanistarast / Newroza Kawa system.

Its purpose is to ensure that:
- all conceptual layers remain ordered
- all machine-readable layers remain consistent
- all scientific layers remain testable
- all future contributors follow a fixed structural logic

6. CONTRIBUTION RULES
Any future file or contribution must satisfy the following conditions:
It must not contradict the canonical concept registry.
It must not violate the class map.
It must not introduce relations absent from the relation map without explicit extension notes.
It must not bypass ontology, validation, or synthesis.
It must preserve falsifiability.
It must remain compatible with graph representation.
