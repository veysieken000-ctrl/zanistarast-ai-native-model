# Tests

This directory contains validation and regression tests for the Zanistarast AI Native Model.

The goal of these tests is to verify that the framework behaves consistently and that future changes do not introduce contradictions.

## Categories

- Deduction tests
- Induction tests
- Rasterast verification tests
- Hallucination detection tests
- Knowledge consistency tests
- Benchmark regression tests

Future versions may include automated execution using pytest or other testing frameworks.
